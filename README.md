# Sneha Kumari — Creative Designer Portfolio

Angular 17 + Tailwind CSS portfolio website.

## Quick Start

```bash
npm install
npm start
```

Then open http://localhost:4200

## Requirements
- Node.js 18+
- npm 9+

## Build for Production
```bash
npm run build
```

## Tech Stack
- Angular 17 (standalone components)
- Tailwind CSS 3
- Google Fonts: Playfair Display, Dancing Script, DM Sans

## Structure
src/app/components/
  navbar/    — Fixed nav, dark mode toggle, mobile menu
  hero/      — Hero section with animated intro
  about/     — About me + stats
  projects/  — Featured projects grid
  services/  — Services offered
  skills/    — Animated skill bars (scroll-triggered)
  contact/   — Contact form with validation
  footer/    — Footer with links
